# UNEP ICT Assignment -- Skills Portal

This project is a web-based UNEP Skills Portal developed using PHP and
MySQL on XAMPP. It allows administrators to manage staff profiles, track
technical skills, assign duty stations, and generate structured staff
listings.

The system includes secure authentication, role-based access control,
and full CRUD functionality for staff records.

------------------------------------------------------------------------

## Technologies Used

-   PHP\
-   MySQL\
-   Apache (XAMPP)\
-   HTML, CSS, JavaScript\
-   Git & GitHub

------------------------------------------------------------------------

## System Features

-   Secure login and logout\
-   Staff profile management\
-   Add, edit, delete, and view staff records\
-   Skills and duty station management\
-   Structured staff listing table\
-   Database backup in SQL format

------------------------------------------------------------------------

## Requirements

-   XAMPP (Apache and MySQL)\
-   Web browser (Chrome, Edge, Firefox)

------------------------------------------------------------------------

## Setup Instructions

1.  Install XAMPP on your computer.\

2.  Start Apache and MySQL from the XAMPP Control Panel.\

3.  Copy the project folder to:

    C:`\xampp`{=tex}`\htdocs`{=tex}`\unep`{=tex}-skills-portal

4.  Open phpMyAdmin:

    http://localhost/phpmyadmin

5.  Create a database named:

    unep_skills

6.  Import the SQL file:

    database/unep_skills.sql

7.  Open the application in your browser:

    http://localhost/unep-skills-portal/
